# The `stack new` command

~~~text
stack new PACKAGE_NAME [--bare] [TEMPLATE_NAME]
          [-p|--param KEY:VALUE] [DIR(S)] [--omit-packages] [--force]
          [--ignore-subdirs]
~~~

This command is described in Stack's introductory [User's Guide](GUIDE.md).
